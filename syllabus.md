# **syllabus.md**  
### **12‑Week Bootstrap, CSS & JavaScript Learning Course**  
Designed for absolute beginners through advanced learners.  
No prior coding experience required.

---

# **📘 Course Overview**
**Length:** 12 weeks  
**Time Commitment:** 3–6 hours per week  
**Technologies:**  
- HTML  
- CSS (Flexbox, Grid)  
- Bootstrap 5  
- Vanilla JavaScript (ES modules)  
- Sass (Bootstrap theming)  
- Vite (build system)  
- GitHub Pages / Netlify / Vercel (deployment)

**Outcome:**  
By Week 12, you will build and deploy a fully responsive, accessible, interactive multi‑page website with a custom Bootstrap theme and production build pipeline.

---

# **📗 Week‑by‑Week Syllabus**

---

## **Week 1 — HTML Foundations**
### **Goals**
Understand basic page structure and semantic HTML.

### **Topics**
- `<!doctype>`, `<html>`, `<head>`, `<body>`
- Headings, paragraphs, links, images, lists
- Semantic elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`

### **Exercises**
- Build a simple “Profile” or “Business” page using semantic HTML.

### **Deliverable**
`index.html` with clean, semantic structure.

---

## **Week 2 — CSS Basics & Layout**
### **Goals**
Learn selectors, box model, typography, and basic layout.

### **Topics**
- Selectors, specificity, external stylesheets  
- Box model, margin, padding, `display`, `position`  
- Colors, fonts, responsive thinking

### **Exercises**
- Style Week 1 page with external CSS.  
- Create a two‑column layout using Flexbox.

### **Deliverable**
Styled `index.html` with a simple responsive layout.

---

## **Week 3 — Bootstrap Setup & Grid System**
### **Goals**
Use Bootstrap via CDN and understand the grid system.

### **Topics**
- Bootstrap CDN  
- Containers, rows, columns  
- Breakpoints (`sm`, `md`, `lg`, `xl`, `xxl`)  
- Spacing utilities (`.m-*`, `.p-*`)

### **Exercises**
- Rebuild layout using Bootstrap grid and utilities.

### **Deliverable**
Bootstrap‑powered responsive layout.

---

## **Week 4 — Core Components & Multi‑Page Site**
### **Goals**
Build a small multi‑page site using Bootstrap components.

### **Topics**
- Navbar, cards, buttons, forms  
- Responsive images  
- Basic accessibility (labels, alt text)

### **Exercises**
- Create `index.html`, `about.html`, `contact.html` with shared navbar/footer.

### **Deliverable**
3‑page responsive site using Bootstrap components.

---

## **Week 5 — Bootstrap JavaScript & Interactivity**
### **Goals**
Initialize Bootstrap JS components and handle basic events.

### **Topics**
- Bootstrap JS bundle  
- `data-bs-*` attributes  
- Modals, dropdowns, tooltips, carousel  
- Vanilla JS event listeners

### **Exercises**
- Add modal “Sign up” and carousel.  
- Write JS to open modal on button click.

### **Deliverable**
Interactive site with working modal and carousel.

---

## **Week 6 — Forms & Client‑Side Validation**
### **Goals**
Implement client‑side validation and better form UX.

### **Topics**
- HTML5 validation (`required`, `type`, `pattern`)  
- Bootstrap validation classes  
- Custom JS validation  
- ARIA error messages

### **Exercises**
- Add validation to contact/signup forms.  
- Show inline error messages and success feedback.

### **Deliverable**
Accessible, validated forms with clear feedback.

---

## **Week 7 — Utilities, Theming & CSS Overrides**
### **Goals**
Use utilities effectively and lightly customize Bootstrap.

### **Topics**
- Spacing, text, background, display utilities  
- Small custom CSS overrides  
- Intro to design tokens (colors, fonts)

### **Exercises**
- Define a simple color/typography system.  
- Replace custom CSS with utilities where possible.

### **Deliverable**
Consistent visual system using utilities + minimal custom CSS.

---

## **Week 8 — Intermediate Project: Landing Page**
### **Goals**
Build a focused product landing page.

### **Topics**
- Hero sections  
- Feature grids  
- CTAs  
- Modal signup  
- Validated form

### **Exercises**
- Create `landing.html` with hero, features, modal, and form validation.

### **Deliverable**
Polished landing page demonstrating components + interactivity.

---

## **Week 9 — Sass, Bootstrap Customization & Vite**
### **Goals**
Customize Bootstrap via Sass and set up a build pipeline.

### **Topics**
- Install Bootstrap via npm  
- `_variables.scss` overrides  
- Vite dev server & build scripts

### **Exercises**
- Create `styles.scss` importing Bootstrap + variables.  
- Run `npm run dev` and `npm run build`.

### **Deliverable**
Custom Bootstrap theme compiled via Sass.

---

## **Week 10 — Advanced Layout & Performance**
### **Goals**
Combine CSS Grid with Bootstrap and improve performance.

### **Topics**
- CSS Grid for complex sections  
- Responsive images (`srcset`, `sizes`)  
- Lazy loading (`loading="lazy"`)  
- `defer`/`async` scripts

### **Exercises**
- Replace one complex section with CSS Grid.  
- Add lazy loading and responsive images.

### **Deliverable**
Site with Grid‑based section + performance improvements.

---

## **Week 11 — Modular JavaScript & State**
### **Goals**
Structure JS as modules and manage simple state.

### **Topics**
- ES modules (`import`/`export`)  
- Component modules (modal, tabs)  
- `localStorage` for theme persistence

### **Exercises**
- Move modal/tabs logic into separate modules.  
- Persist theme choice across reloads.

### **Deliverable**
Modular JS structure with persistent theme.

---

## **Week 12 — Capstone Project & Deployment**
### **Goals**
Finish and deploy a multi‑page site.

### **Topics**
- Accessibility review  
- Lighthouse audits  
- Deployment (GitHub Pages / Netlify / Vercel)

### **Exercises**
- Fix accessibility issues (keyboard, labels, alt text).  
- Run Lighthouse and address major issues.  
- Deploy production build.

### **Deliverable**
Live, deployed multi‑page site with custom Bootstrap theme.

---

# **📙 Final Capstone Requirements**
Your final project must include:

- Custom Bootstrap theme via Sass  
- Multi‑page layout  
- Responsive design  
- Accessible components  
- Client‑side validation  
- Modular JavaScript  
- Lazy loading + responsive images  
- Production build via Vite  
- Deployment to GitHub Pages / Netlify / Vercel
