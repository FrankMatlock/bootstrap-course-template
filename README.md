# Bootstrap CSS & JavaScript 12‑Week Course Template

This repository contains the starter files, syllabus, checklist, and build system for a complete 12‑week learning course covering HTML, CSS, Bootstrap, JavaScript, Sass, and Vite.

## 📚 Course Contents
- `syllabus.md` — full 12‑week learning plan  
- `checklist.md` — project milestones and rubric  
- `src/` — HTML, SCSS, JS starter files  
- `package.json` — Vite build scripts  
- `vite.config.js` — build configuration
